## Project Structure

  - `code/playground.ipynb` - Main notebook
  - `code/prompts/` - Jinja templates for prompts

## Website

The website is built using the [Academic Project Page Template](https://github.com/eliahuhorwitz/Academic-project-page-template). To view locally, open `website/index.html` in a browser.

To customize:
1. Edit `website/index.html` and replace TODO placeholders with your paper details
2. Add your images/videos to `website/static/images/`, `website/static/videos/`
3. Add paper PDF to `website/static/pdfs/`
4. Replace favicon in `website/static/images/favicon.ico`
